# Design Goal

Reference screenshot: `assets/reference/design-target.png`

This project is targeting the HSL iPhone ticket screen shown in the reference:

- iPhone-style status bar and dynamic island.
- Top tabs with "Your tickets" selected.
- White season-ticket card on a light gray app background.
- QR code, ticket number, subscription badge, validity, customer group, and AB zones panel.
- Fixed bottom navigation with Tickets selected.

The first implementation is a static HTML/CSS mockup. Future app work can use this page as the visual baseline for component structure, spacing, colors, and typography.
